## Using Cordova.
Features:
* List WiBox folders
* Thumbnail for image files.
* Icons for known file types.
* Play video files including .mp4, .mov, .webm, etc.
* Play audio files including .mp3, .m4a, .wav, etc.
* Many more to come...
